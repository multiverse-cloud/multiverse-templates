# Purple Wave

> A bold, animation-heavy designer portfolio built on purple gradients, floating blobs, and smooth motion.

## Live Demo

[View Live Demo](https://ethhandy.github.io/portfolio-zoo/purple-wave/)

## Features

- Animated gradient blobs floating in background with parallax on scroll
- Mesh grid overlay with radial mask
- Rotating orbital rings with glowing dots in the hero
- Pulsating gradient core orb
- Staggered word-by-word hero title animation
- Infinite skills ticker marquee
- Project cards with 3D tilt effect on mouse move
- Animated gradient skill bars that fill on scroll
- Floating stat badges with gentle bobbing animation
- Scroll reveal with staggered sibling timing
- Glassmorphic tags and buttons with glow shadows
- Fully responsive

## How to Customize

1. **Your info** — Edit names, role, bio, stats, and project details in `index.html`
2. **Colors** — Change the purple palette in `:root` in `style.css` (swap purples for blues, reds, etc.)
3. **Projects** — Replace project cards with your own work, swap gradient backgrounds for real images
4. **Skills** — Edit the `data-width` attributes on `.skill-fill` elements to set percentages
5. **Services** — Update the service cards with your own offerings
6. **Contact** — Replace email and social links with your own

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no frameworks)
- Google Fonts (Outfit, Sora)
