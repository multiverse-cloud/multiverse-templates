# macOS Desktop

> A macOS-inspired desktop portfolio with draggable Finder windows, a dock, and a working terminal aesthetic.

## Features

- macOS desktop environment with menu bar, dock, and draggable windows
- Finder-style navigation with sidebar
- Terminal window with command-line aesthetic
- Contact form with send animation
- Smooth window open/close/minimize/maximize animations
- Fully responsive — stacks cleanly on mobile

## How to Customize

1. **Your info** — Open `index.html` and replace the placeholder name, bio, role, and stats in the About window
2. **Experience** — Edit the experience items in the `win-experience` section
3. **Skills** — Update skill categories and pills in the `win-skills` section
4. **Terminal** — Customize the fake terminal commands and output in `win-terminal`
5. **Contact** — Wire up the contact form to your preferred service (Formspree, EmailJS, etc.)
6. **Colors** — Edit the CSS variables at the top of `style.css` (`:root` block) to change the entire color scheme
7. **Meta tags** — Update the Open Graph and Twitter Card meta tags with your own info

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no frameworks)
- Google Fonts (Inter, JetBrains Mono)
