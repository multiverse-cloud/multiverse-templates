# Designer Folio

> A minimal, eye-catching portfolio for UX/UI designers with bold typography, smooth scroll reveals, and a neon-lime accent.

## Live Demo

[View Live Demo](https://ethhandy.github.io/portfolio-zoo/designer-folio/)

## Features

- Bold hero with outlined display type and hover fill effect
- Cursor-following radial glow
- Infinite scrolling marquee with skills
- Project cards with gradient thumbnails and hover lift
- Scroll reveal animations with staggered timing
- Pill-style skill tags with hover glow
- Film grain texture overlay
- Responsive across all breakpoints

## How to Customize

1. **Your info** — Edit names, role, bio, and stats directly in `index.html`
2. **Projects** — Replace the project cards with your own case studies, swap gradient backgrounds for real screenshots
3. **Colors** — Change `--accent` in `style.css` `:root` to any color (try `#6366f1`, `#f43f5e`, `#06b6d4`)
4. **Contact links** — Update email, Dribbble, LinkedIn, Twitter with your own
5. **Skills marquee** — Edit the marquee spans to list your own skills/tools

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no frameworks)
- Google Fonts (DM Sans, Space Grotesk)
